# Flex

<Status />

O `apollo-flex` é um componente que renderiza uma div com `display: flex` e possui propriedades de estilo abaixo.

<Story id="layouts-flex--base" />

## Spacer

Você pode utilizar uma propriedade `spacer` para cria um espaçamento igual entre os elementos. Mantendo o primeiro grudado no início e o último no final.

<Story id="layouts-flex--spacer" />

## Direction

Passe a propriedade `direction` para alterar a direção do filho.

<Story id="layouts-flex--direction" />

## Reverse

Pode mudar a ordem usando `reverse`.

<Story id="layouts-flex--reverse" />

## Alignment

Passe a propriedade `align-items` para alterar o alinhamento do filho ao longo do eixo transversal.

<Story id="layouts-flex--alignment" />

## Justify

Passe a propriedade `justify` para alterar o alinhamento do filho ao longo do eixo principal.

<Story id="layouts-flex--justify" />

## Wrap

Passe a propriedade `wrap` para quebrar linha automaticamente quando não couber mais filhos.

<Story id="layouts-flex--wrap" />
