# Box

<Status />

O componente `apollo-box` serve como um componente encapsulador (wrapper) para a auxiliar na maioria das necessidades de uso com CSS. Use o componente como um layout primitivo para adicionar margem, preenchimento e cores ao conteúdo.

O componente `apollo-box` compõe todas as funções de estilo que são expostas no [design-system-tokens](https://github.com/PicPay/picpay-frontend/packages/697883).

<Story id="layouts-box--base" />

Segue abaixo as principais propriedades do componente. Para visualização de todas as propriedades acesse a aba `Notes`.

Passe a propriedade `bg` para alterar o fundo e color para alterar a cor da fonte.
