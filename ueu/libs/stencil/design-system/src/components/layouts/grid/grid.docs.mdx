## Spanning columns

Em alguns layouts, você pode precisar de certos itens de grade para abranger uma quantidade específica de colunas ou linhas em vez de uma distribuição uniforme.

Para conseguir isso, você precisa passar o `colSpan` ao componente `apollo-grid-item` para se estender pelas colunas e também passar o `rowSpan` ao componente `apollo-grid-item` para se estender pelas linhas. Você também precisa especificar o `templateColumn` e `templateRows`.

## Starting and ending lines

Passe as propriedades `colStart` e o `colEnd` para o componente `apollo-grid-item` para fazer um elemento começar ou terminar na posição da grade.
