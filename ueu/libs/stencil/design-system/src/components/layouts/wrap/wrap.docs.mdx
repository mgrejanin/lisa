# Wrap

<Status />

O `apollo-wrap` é um componente de layout que adiciona um espaço definido entre seus filhos.
Ele quebra a linha automaticamente se não houver espaço suficiente para caber mais filhos na mesma linha.

<Story id="layouts-wrap--base" />

<Props id="layouts-wrap--base" />

## Spacing

Passe a propriedade `spacing` para aplicar espaçamento consistente entre cada filho.

<Story id="layouts-wrap--spacing" />

## Alignment

Passe a propriedade `align-items` para alterar o alinhamento do filho ao longo do eixo transversal.

<Story id="layouts-wrap--alignment" />

## Justify

Passe a propriedade `justify` para alterar o alinhamento do filho ao longo do eixo principal.

<Story id="layouts-wrap--justify" />

## Direction

Passe a propriedade `direction` para alterar a direção do filho.

<Story id="layouts-wrap--direction" />
