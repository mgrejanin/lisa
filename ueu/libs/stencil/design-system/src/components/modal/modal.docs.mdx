## Events

| Event        | Description                                                   | Type                      |
| ------------ | ------------------------------------------------------------- | ------------------------- |
| `closeModal` | Evento disparado quando o botão de fechar o modal for clicado | `CustomEvent<MouseEvent>` |

### Orientações:

Para ter acesso ao valor do evento emitido deve ser usada a propriedade `detail` do objeto `CustomEvent` que é emitido nos métodos.

## Methods

### `close() => Promise<void>`

API para fechar o modal

#### Returns

Type: `Promise<void>`

### `open() => Promise<void>`

API para abrir o modal

#### Returns

Type: `Promise<void>`

### Orientações para angular:

Para utilizar os métodos do componente utilize o @ViewChild() conforme exemplo a seguir:

Html:

```html
<apollo-button (onClick)="open()">Show modal</apollo-button>
<apollo-modal>
    <p>Insira seu conteúdo aqui</p>
</apollo-modal>
```

Ts:

```javascript
@ViewChild('modal') modal: ApolloModal;

//...

open() {
    this.modal.open();
}
```
