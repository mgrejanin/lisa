## Events

| Event          | Description                                        | Type                   |
| -------------- | -------------------------------------------------- | ---------------------- |
| `apolloBlur`   | Evento disparado quando o componente perde o foco  | `CustomEvent<null>`    |
| `apolloChange` | Evento disparado quando o valor do componente muda | `CustomEvent<boolean>` |

### Orientações:

Para ter acesso ao valor do evento emitido deve ser usada a propriedade `detail` do objeto `CustomEvent` que é emitido nos métodos.

## Methods

### `setChecked(checked: boolean) => Promise<void>`

API que modifica o valor do componente

#### Returns

Type: `Promise<void>`

### `setDisabled(disabled: boolean) => Promise<void>`

API que habilita/desabilita o componente

#### Returns

Type: `Promise<void>`

### Orientações para angular:

Para utilizar os métodos do componente utilize o @ViewChild() conforme exemplo a seguir:

Html:

```html
<apollo-switch-toggle #switch />
```

Ts:

```javascript
@ViewChild('switch') switch: ApolloSwitch;

//...

setChecked() {
    this.switch.setChecked(true);
}
```
