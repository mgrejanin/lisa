# Sidebar

<Status variant="inDevelopment" />

Uma navegação lateral vertical fácil de integrar, ideal para painéis e dashboard. Nossa barra lateral de navegação pode tornar sua vida mais fácil, fornecendo um modelo inicial simples para seu próximo projeto.

Para iniciar, usamos três componentes: `<apollo-sidebar>` que atua como um contêiner estrutural para nosso conteúdo de navegação, `<apollo-nav-item>` que representa o menu principal e `<apollo-collapse>` que representa o conteúdo expansivo.

## Events

| Event             | Description                                                    | Type               |
| ----------------- | -------------------------------------------------------------- | ------------------ |
| `apolloCollapsed` | Evento disparado quando o componente é alternado abert/fechado | `CustomEvent<any>` |

### Orientações:

Para ter acesso ao valor do evento emitido deve ser usada a propriedade `detail` do objeto `CustomEvent` que é emitido nos métodos.
