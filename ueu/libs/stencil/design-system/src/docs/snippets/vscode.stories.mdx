# Snippets

Esta extensão para Visual Studio Code adiciona snippets para Angular, React, TypeScript e HTML.

## Uso

Basta digitar as letras `apl` para obter uma lista de todos os snippets de componentes **Apollo Design System** disponíveis.

Digite parte de um snippet, pressione `enter` e o snippet se desdobra.

Como alternativa, pressione `Ctrl` + `Space`(Windows, Linux) ou `control` + `Space`(macOS) para ativar trechos de dentro do editor.

Abaixo está uma lista de todos os snippets disponíveis.
