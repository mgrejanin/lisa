Identifique o nome do projeto A identificação do projeto (promoção, campanha, bonificação, etc) ajudará apenas na organização das credenciais dentro do Painel Lojista, ela não configurará a promoção/campanha. Após esse passo, uma credencial (Client_ID e Client_Secret) serão criadas e você deve usá-las para configurar sua integração, consulte a documentação da API. Atenção: o formato do site precisa ter https:// ou http:// no endereço.
