Vantagens do Open Banking

Quem se integrar com nossas APIs vai poder levar o PicPay para o seu negócio com segurança e praticidade. Esse novo modelo foi pensado para dar mais autonomia ao consumidor, através do compartilhamento de dados financeiros pelos clientes entre as instituições financeiras participantes.

Segurança

Toda a comunicação via API será criptografada seguindo os padrões impostos pelo Banco Central. Além disso, na construção das nossas APIs, o PicPay está sempre adequado ao certificado digital ICP-Brasil. Isso significa que toda integração é autenticada de ponta a ponta, garantindo segurança reforçada aos nossos usuários.

APIs do Open Banking

Cada banco, empresa, fintech ou operadora terá suas APIs públicas. Isso possibilita criar um ecossistema seguro e escalável de troca de dados das operações financeiras do mercado brasileiro entre instituições participantes do Open Banking. Nenhuma instituição terá acesso aos dados de um cliente sem que ele antes permita esse compartilhamento (exceto pelos dados não sensíveis, como nome e data de nascimento, por exemplo).

Para conhecer as APIs de Open Banking do Picpay, acesse a nossa página de documentações.
