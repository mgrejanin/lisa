## O que é Remessa de Créditos com PicPay?

É uma API que permite distribuição de dinheiro e créditos para usuários do app. De forma rápida e eficaz, você pode realizar integrações com o serviço e configurar as validações necessárias para o seu contexto.
É mais segurança e fluidez na entrega dos valores, com todos os mecanismos necessários para monetizar prêmios, de forma regulamentada. O usuário ainda pode utilizar o benefício como forma de pagamento de contas, créditos no celular ou no cartão de transporte e com compras no PicPay.

## Começar a usar é simples

Você faz um cadastro no nosso Painel Lojista, usa uma chave de integração no seu e-commerce e configura a aplicação em poucos passos.

1. Após abrir o aplicativo, tocar no botão 'Pagar' na parte inferior da tela

2. No campo de busca na parte superior da tela, procurar seu perfil de assinaturas

3. Depois de escolher o plano, é só confirmar a assinatura

## Apenas uma tarifa para implantação

A forma de pagamento é tão simples quanto a configuração da API! É cobrada apenas uma taxa sobre o valor total a ser transacionado.
